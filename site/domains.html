Domain Overlays

Every artifact ships with eight domain overlays. The overlay tells you what changes for that industry — added risk categories, required signatories, new cost lines, or explicit boundaries — without rewriting the artifact.

The eight overlays

How to use an overlay

  1. Complete the base artifact using the DOCX template as if the engagement were generic.
  2. Open the overlay section at the bottom of that artifact.
  3. Add the overlay-specific rows to the pricing table, risk register, or decision log as instructed.
  4. Have the domain-specific signatory review before freezing the artifact.

Adding a new domain

To add a domain (e.g. healthcare/HIPAA), add a new entry to window.ACA.domains in site/assets/data.js with a key, name, tagline, and add description. Then add a domain-specific section to each artifact template where the overlay materially changes the answer.
